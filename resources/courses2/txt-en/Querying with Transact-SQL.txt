Computer Science Transact-SQL is an essential skill for database professionals and developers working with SQL databases. This course takes you from your first SELECT statement to implementing transactional programmatic logic through a combination of expert instruction, demonstrations, and practical labs.   The course consists of multiple modules, each exploring a key area of the Transact-SQL language, with a focus on querying and modifying data in Microsoft SQL Server or Azure SQL Database. The labs are based on a sample database that you can deploy easily in Azure SQL Database, enabling you to get hands-on experience with Transact-SQL without any need to install and configure a database server.
